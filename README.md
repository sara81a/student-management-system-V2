**Student Management System**
This is a web application built using Java, Spring Boot, Thymeleaf, and MySQL. The application provides functionality for managing student records in a database.

**Features**
View a list of all students.
Add a new student with full information:
First Name
Last Name
Email
Edit student information.
Delete a student.
View detailed information about a student.

**Steps to Run the Application**
**Prerequisites**
Before running the application, ensure you have the following installed on your system:
Docker (to build and run the application using containers).
Git (to clone the repository).
**Setup**

Clone the repository:

git clone https://github.com/<your-username>/student-management-system.git
cd student-management-system

**Run the Application**
Build the Docker images:
docker-compose build

Start the application:
docker-compose up

Access the application in your browser:
http://localhost:8080/students
